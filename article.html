1. 对于变量的概念，以及在c语言中如何通过终端打印（输出）出我们想要输出的内容

在c语言和我们后面会学到的其他语言中，数据类型大致可以分为一下几种： 1.整数类型（整型） 2.小数类型（浮点型）又称之为浮点数类型 3.字符类型，主要用于存储字母和单个符号 

#include <stdio.h>
int main() {
    int a = 11; // 整数类型
    char b = 'N'; // 字符类型
    double a = 1.1; // 小数类型
}

这是我们在初始学习中会比较常用到的数据类型，在每一行的开头都是，数据类型的名字，比如：int。 接着后面写的就是变量的名字，这些是你自己所定义的想写什么就写什么，等于号后面就是，你想赋给他的数据类型。 需要注意的就是在做字符类型赋值时需要用到单引号包住字母或符号。

对于在c语言中打印内容，我们会先学习用到printf

#include <stdio.h>
int main() {
    printf("%d\n", a);
}
