在完成初步设置之后，我们则直接进入c语言的学习

在c语言中我们经常能在代码开头看到include这样的字眼，这是用来引用函数库的（功能类似于在其他语言中的import）因为在c语言中他本身不包含任何函数， 他本身只包含语法（if for while还有逻辑运算符），像printf和scanf这样的函数都是存在函数库了的如果我们需要使用他们就必须引用对应的函数库， 而在我们刚开始学习c语言时会经常用到<stdio.h>就是因为他里面包含这printf和scanf。

对于在c语言中打印内容，我们会先学习用到printf

#include <stdio.h>
int main() {
    printf("Hello World!\n");
}

printf函数的用法在括号中有两段内容printf("",), 分别是逗号前带引号的部分和逗号后不带引号的部分，逗号后不带引号的部分我们将在本章节后半部分“变量的相关内容”中 讲解，在引号中的内容就是我们要打印的内容了， 就像上面的实例一样，'\n'这个东西是我们在c语言中会经常用到的换行符号，他的作用是在他的位置做一个换行然后接着输出后面的东西，就像这样

#include <stdio.h>
int main() {
    printf("Hello\nWorld!\n");
}

对于变量的概念，以及在c语言中如何通过终端打印（输出）出我们想要输出的内容

在c语言和我们后面会学到的其他语言中，数据类型大致可以分为一下几种：

1.整数类型（整型）

2.小数类型（浮点型）又称之为浮点数类型

3.字符类型，主要用于存储字母和单个符号

这是我们在初始学习中会比较常用到的数据类型，在每一行的开头都是，数据类型的名字，比如：int。 接着后面写的就是变量的名字，这些是你自己所定义的想写什么就写什么，等于号后面就是，你想赋给他的数据类型。 需要注意的就是在做字符类型赋值时需要用到单引号包住字母或符号。

#include <stdio.h>
int main() {
    int a = 11; // 整数类型
    char b = 'N'; // 字符类型
    double a = 1.1; // 小数类型
}

我们在前面的章节中提到，printf("",)打印这个功能还有逗号后的内容，那printf逗号后的部分其实就是用来写变量的，我们把变量的名字写在逗号后面，在 引号里面写上的你变量类型对应的占位符，就可以答应出想要的变量了,就像这样

#include <stdio.h>
int main() {
    int a = 11; // 整数类型
    printf("%d\n", a);  // %d就是整数类型的占位符
}

占位符，顾名思义就是告诉程序我们会在这里打印一个变量，但是我们现在不知道他的值是多少，只知道他是什么类型，所以用一个占位符告诉程序我们要打印什么类型的变量，让他去逗号后面找， 每个类型都有对应的占位符，类型根据你对变量的定义时的类型来决定。
